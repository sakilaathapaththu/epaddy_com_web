Project Scope

Revolutionary Agriculture

The ePaddy project in Sri Lanka is a groundbreaking initiative that uses advanced technologies to support sustainable paddy cultivation. It uses Convolutional Neural Networks (CNN) to identify diseases, pest infestations, and crop growth stages, and Gradient Boosting algorithms for weather-based disease forecasting. An IoT-powered water management system uses ESP8266 microcontrollers and ultrasonic sensors for optimal irrigation, reducing water usage. Real-time soil quality monitoring is done using NPK sensors, which feed data to a machine learning model for crop recommendations. The solution is accessible through a user-friendly mobile application built with Flutter and connected to Firebase. This aims to increase productivity, enhance sustainability, and empower farmers with modern tools tailored to their needs.

Core Problem Statement

"Sri Lankan paddy farmers face challenges like late detection of diseases and pests, poor soil and water management, and limited access to timely information and modern farming methods. These issues lead to low harvests and reduced productivity. Therefore, a smart and user-friendly system is needed to help farmers detect problems early, receive proper treatments, and manage their fields more effectively."

Methodology

Phase 1: Research & Analysis

Conducted detailed literature review and field research to identify problems in Sri Lankan paddy farming. Collected soil, disease, pest, and climate datasets for training ML models.

Phase 2: System Design

Designed a modular system integrating mobile app (Flutter), AI (CNN, Gradient Boosting), and IoT sensors (ESP8266, ultrasonic, NPK). Established Firebase for real-time data and storage.

Phase 3: Development

Implemented ML models for disease/pest detection, soil-based crop recommendation, and harvest stage prediction. Developed APIs using Flask and frontend with Dart + Flutter.

Phase 4: Integration & Testing

Integrated all modules. Conducted functional testing in paddy fields. Verified real-time sensor readings, ML outputs, and mobile usability with farmers and experts.

Phase 5: Deployment & Evaluation

Deployed the system in real environments. Collected user feedback and performance data. Final adjustments were made to improve prediction accuracy, UI/UX, and irrigation response logic.
